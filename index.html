Quais são as peças de um computador?

Como funciona um computador?

Unidade Central de Processamento (CPU)

A CPU é o cérebro do computador e executa instruções.

Memória RAM

A RAM armazena temporariamente dados e programas em execução.

Armazenamento

O armazenamento (disco rígido ou SSD) guarda dados permanentemente.

Placa-Mãe

A placa-mãe é o circuito principal que conecta todos os componentes do computador.

Placa de Vídeo (GPU)

A placa de vídeo, ou GPU (Graphics Processing Unit), é responsável pelo processamento gráfico.
