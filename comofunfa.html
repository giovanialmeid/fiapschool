Como funciona desde que você liga o computador?

Quais são as peças de um PC?

1. Ligando o Computador

Quando o botão de energia é pressionado, a placa-mãe recebe energia e inicia o processo de inicialização.

2. POST(Power-On Self-Test)

A placa-mãe realiza um teste de verificação para garantir que todos os componentes essenciais, como CPU, RAM e dispositivos de armazenamento, estejam funcionando corretamente.

3. BIOS

O BIOS (Basic Input/Output System) é carregado da memória ROM da placa-mãe e fornece instruções de baixo nível para o hardware.

4. Iniciando o Sistema Operacional

A placa-mãe localiza o dispositivo de inicialização (geralmente um HD ou SSD) e carrega o sistema operacional na memória RAM.

5. Login do usuário

O sistema operacional inicia e exibe a tela de login. O usuário faz o login usando sua senha ou identificação biométrica.

6. Área de trabalho

Após o login, o sistema operacional carrega a área de trabalho do usuário, exibindo ícones e permitindo a interação.

7. Abrindo os programas

O usuário pode abrir programas clicando em ícones ou usando comandos. Quando um programa é aberto, ele é carregado na RAM a partir do HD ou SDD.

8. Executando o programa

O programa em execução na RAM interage com o sistema operacional e outros componentes para realizar tarefas específicas, como navegar na web, editar documentos ou jogar jogos.

9. Ultilizando os componentes para executar.

O programa pode acessar recursos como CPU, GPU, RAM e dispositivos de armazenamento para funcionar corretamente.
